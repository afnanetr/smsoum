## 🎵 MUSIC FILE NEEDED:
You need to add a music file for the lightstick.
Supported formats: .mp3, .ogg, .wav

Example filename: "bini_song.mp3"

To add it:
1. Place your music file in the game folder
2. Open script.js
3. Find line ~100 where the game starts (startBtn.addEventListener)
4. Add this line:
   initLightstickMusic("bini_song.mp3");
