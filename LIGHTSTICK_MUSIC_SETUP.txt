🎵 LIGHTSTICK MUSIC SETUP INSTRUCTIONS
=====================================

To add music to the lightstick:

1. Add your music file to the game folder
   Example: "bini_song.mp3" or "bini_song.ogg"

2. Open script.js and find this line (near the end of the file):
   // Call this with your music file: initLightstickMusic("your_music_file.mp3");

3. Add this line right after the startBtn click event (around line 100):
   initLightstickMusic("bini_song.mp3");
   
   Replace "bini_song.mp3" with your actual music filename.

Example:
--------
startBtn.addEventListener("click", () => {
    menu.style.display = "none";
    game.style.display = "block";
    
    // Initialize lightstick music
    initLightstickMusic("bini_song.mp3");  // <-- ADD THIS LINE
    
    showMessage(
        "Movement",
        "Use WASD, ZQSD, or the Arrow Keys to move around!"
    );
    
    initializeJournal();
});

=====================================

HOW THE LIGHTSTICK WORKS:
- Lightstick is LOCKED and grayed out at the start
- Complete Mission 2 (collect all garden pieces) to unlock it
- Once unlocked, DOUBLE-CLICK the lightstick to turn it on
- Music will play when it's on
- Double-click again to turn it off

LIGHTSTICK LOCATION:
- Fixed position at: left: 900px, top: 200px
- You can change this in style.css under .lightstick-placed
